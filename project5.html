A Cryptocurrency Explorer & Market Analyzer

In this project, I developed a cryptocurrency explorer and market analyzer that processes large-scale crypto data efficiently and extracts meaningful insights. The tool is designed to handle big datasets without overloading memory, using smart file-reading and in-place data processing techniques.

The analyzer supports key market functions such as:

  • Calculating price statistics over custom intervals
  • Sorting and comparing cryptocurrencies
  • Identifying peak values
  • Finding the closest prices to a target

To deepen market understanding, I also implemented a method to uncover correlations between cryptocurrencies using graph-based algorithms like Prim’s and Breadth-First Search (BFS).

From raw data to structured insights, the project integrates algorithmic logic with practical use cases—making it a valuable tool for traders, researchers, and anyone interested in crypto market behavior.
